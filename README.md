## üïπÔ∏è How to Play

1. Player X starts first.
2. Click on any empty cell to place your marker.
3. The first player to get 3 markers in a row (horizontal, vertical, or diagonal) wins!
4. If all cells are filled and no one has 3 in a row, it's a draw.
5. Click the "Restart Game" button to play again.
